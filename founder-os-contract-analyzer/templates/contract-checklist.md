# Contract Review Checklist

Use this checklist to ensure comprehensive contract analysis. Check each item during review.

## 1. Parties & Identification
- [ ] All parties clearly identified with legal names
- [ ] Correct entity types specified (LLC, Corp, individual, etc.)
- [ ] Contact information or registered addresses included
- [ ] Signing authority confirmed (authorized representative)

## 2. Scope of Work / Services
- [ ] Deliverables clearly defined and measurable
- [ ] Acceptance criteria specified
- [ ] Change order / scope change process defined
- [ ] Revision limits stated (if applicable)
- [ ] Timeline or milestones documented

## 3. Payment Terms
- [ ] Total compensation or rate clearly stated
- [ ] Payment schedule defined (milestone, monthly, upon completion)
- [ ] Currency specified
- [ ] Late payment penalties or interest rates included
- [ ] Expense reimbursement terms defined
- [ ] Invoicing requirements specified

## 4. Intellectual Property
- [ ] IP ownership clearly assigned
- [ ] Pre-existing IP carved out and listed
- [ ] License grants specified (scope, duration, exclusivity)
- [ ] Work-for-hire status clarified
- [ ] Moral rights addressed (if applicable)
- [ ] Open source usage policy stated

## 5. Confidentiality
- [ ] Confidential information defined
- [ ] Standard exclusions included (public info, prior knowledge, independent development)
- [ ] Duration / survival period specified
- [ ] Return or destruction obligations stated
- [ ] Permitted disclosures defined (legal requirement, employees)

## 6. Liability & Indemnification
- [ ] Liability cap specified (dollar amount or contract value multiplier)
- [ ] Mutual vs. one-sided indemnification noted
- [ ] Consequential damages exclusion present
- [ ] Insurance requirements specified (if any)
- [ ] Force majeure clause included

## 7. Termination
- [ ] Termination for cause conditions defined
- [ ] Termination for convenience terms stated
- [ ] Notice period specified
- [ ] Cure period for breach included
- [ ] Payment for work completed upon termination
- [ ] Post-termination obligations listed (IP transfer, data return)

## 8. Dispute Resolution
- [ ] Governing law / jurisdiction specified
- [ ] Dispute resolution method defined (negotiation -> mediation -> arbitration/litigation)
- [ ] Arbitration rules specified (if applicable)
- [ ] Forum / venue stated
- [ ] Fee allocation for dispute resolution

## 9. Non-Compete & Non-Solicitation
- [ ] Duration reasonable (< 1 year preferred)
- [ ] Geographic scope defined and reasonable
- [ ] Industry/activity scope narrow and specific
- [ ] Carve-outs for existing relationships
- [ ] Compensation during non-compete period (if applicable)

## 10. General / Miscellaneous
- [ ] Entire agreement clause present
- [ ] Amendment process defined (written, signed by both)
- [ ] Assignment restrictions stated
- [ ] Severability clause included
- [ ] Notices provision (how to send formal notices)
- [ ] Survival clause identifies which sections survive termination
- [ ] Signature blocks for all parties with dates
